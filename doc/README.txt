This app allows for the saving of information on various subscription services.

There are two screens, the create/edit screen and the list of subscriptions screen.

The subscription list screen allows for the creation or editing of a subscription, and passes the subscription if there was an edit.

The edit screen populates the fields if required and allows the user to input the information about there subscription. When complete the entered informaton gets validated and then returned to the subscription viewing screen.
